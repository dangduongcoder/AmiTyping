## Giới thiệu chung
**AmiTyping** là một dự án nhỏ của mình hiển thị nhân vật PTIT Ami lên màn hình, nhân vật sẽ gõ phím theo như những gì người dùng gõ.

---

## Tải xuống & Cài đặt
📦 **Windows**: [Tải bản Windows](https://github.com/dangduongcoder/AmiTyping/releases/tag/AmiTyping)

Để chạy AmiTyping từ mã nguồn, hãy làm theo các bước sau:

1. **Clone repository:**
   ```bash
   git clone https://github.com/yourusername/AmiTyping.git
   cd AmiTyping
   ```

2. **Cài đặt thư viện cần thiết:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Khởi chạy ứng dụng:**
   ```bash
   python main.py
   ```
